PCManX

GTK+ 2.x 開發的自由 BBS 連線軟體

取得方式

  • Debian 加入 deb http://apt.debian.org.tw unstable main 到 sources.list 中，並於 apt-get update 後執行： apt-get install pcmanx-gtk2
  • Fedora yum install pcmanx-gtk2
  • Mandriva urpmi pcmanx-gtk2
  • Ubuntu apt-get install pcmanx-gtk2
  • openSUSE zypper install pcmanx-gtk2

授權方式

GNU General Public License (GPL)

編譯需求

  • GTK+ 2.4 以上，以 Debian 為例： apt-get install libgtk2.0-dev
  • Xft 2.x，以 Debian 為例： apt-get install libxft-dev
  • Intltool，以 Debian 為例： apt-get install intltool intltool-debian
  • Automake 1.10，以 Debian 為例： apt-get install automake
  • libtool，以 Debian 為例： apt-get install libtool

注意：後兩者只有從 Git 取出的開發版本需要，一般的發行版本只需要前三者

編譯方式

  1. 開發中的功能與選項，請閱讀 README
  2. ./autogen.sh (只有從 Subversion 取出的開發版本需要)
  3. ./configure --help 查看編譯選項，因為功能持續開發中，項目隨時可能增減
  4. ./configure [編譯選項，請參閱上行]
  5. make; make install
